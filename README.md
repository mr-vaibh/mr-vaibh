**[park](https://github.com/mr-vaibh/park)** — Pause a process and free its TCP port. Resume later, same PID and memory intact. Ctrl+Z for servers.

Uses ptrace (Linux) and Mach thread injection (macOS) to inject `close(fd)` into running processes. No recompilation, no LD_PRELOAD.
